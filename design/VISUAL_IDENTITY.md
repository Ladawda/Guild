# Contractie Visual Identity

## Logo Concept

### Primary Logo
The Contractie logo combines a **location pin** with a **house silhouette** to create a distinctive mark that immediately communicates "local home services."

**Concept Description:**
- A rounded location pin shape with the point facing downward
- Inside the pin, a simplified house icon with a peaked roof
- The negative space between the roof and pin creates a subtle "checkmark" shape, suggesting verification
- The house element is slightly offset upward, creating a sense of "home" rising from the location

**Logo Variants:**
- **Full Logo:** Icon + "Contractie" wordmark (horizontal and vertical lockups)
- **Icon Only:** For favicon, app icon, map markers
- **Wordmark Only:** For contexts where the icon is already present

**Clear Space:**
- Minimum clear space around logo: height of the "C" in Contractie
- Never distort, rotate, or apply effects (shadows, outlines) to the logo

## Typography

### Font Families

**Primary Font: Inter**
- Clean, modern, highly legible
- Excellent for UI elements and body text
- Variable font for weight flexibility

**Display/Headings: Inter (Bold/ExtraBold weights)**
- Same family, heavier weights for impact
- Maintains consistency while creating hierarchy

**Fallback Stack:**
```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
```

### Type Scale

| Element | Size | Weight | Line Height | Letter Spacing |
|---------|------|--------|-------------|----------------|
| **H1** | 48px / 3rem | 800 (ExtraBold) | 1.1 | -0.02em |
| **H2** | 36px / 2.25rem | 700 (Bold) | 1.2 | -0.01em |
| **H3** | 28px / 1.75rem | 700 (Bold) | 1.3 | 0 |
| **H4** | 24px / 1.5rem | 600 (SemiBold) | 1.4 | 0 |
| **H5** | 20px / 1.25rem | 600 (SemiBold) | 1.4 | 0 |
| **H6** | 18px / 1.125rem | 600 (SemiBold) | 1.4 | 0 |
| **Body Large** | 18px / 1.125rem | 400 (Regular) | 1.6 | 0 |
| **Body** | 16px / 1rem | 400 (Regular) | 1.6 | 0 |
| **Body Small** | 14px / 0.875rem | 400 (Regular) | 1.5 | 0 |

### Typography Patterns

**Headings:**
- Use ExtraBold (800) for page titles and hero text
- Use Bold (700) for section headings
- Tight letter-spacing (-0.02em to -0.01em) for headlines
- Near Black (`#1F2937`) color

**Body Text:**
- Regular (400) weight for readability
- 16px base size with 1.6 line height
- Dark Gray (`#4B5563`) for secondary content

**UI Elements:**
- SemiBold (600) for buttons and labels
- Medium (500) for captions and metadata
- All-caps with 0.02em letter-spacing for small labels

## Imagery Style

### Photography

**Style Guidelines:**
- **Authentic, not stocky** — Real contractors at work, real homes, real projects
- **Natural lighting** — Avoid harsh studio lighting; prefer daylight and practical lighting
- **Diverse representation** — Show variety in age, ethnicity, gender across contractors and homeowners
- **Action-oriented** — Show people doing the work, not just posing
- **Local context** — Include recognizable neighborhood elements when possible

**Subject Matter:**
- Contractors: Working on projects, consulting with homeowners, loading/unloading tools
- Homes: Before/after shots, work in progress, completed projects
- Details: Tools, materials, craftsmanship close-ups
- Community: Neighborhoods, local landmarks, streetscapes

**Treatment:**
- Slight warmth in color grading (+5% yellow)
- Moderate contrast for depth
- Subtle vignette on hero images
- Avoid heavy filters or extreme color manipulation

### Illustrations

**Style:**
- Clean, friendly line illustrations with flat color fills
- Rounded corners and soft edges
- Limited color palette (primaries + neutrals)
- Simple, recognizable shapes

**Usage:**
- Empty states
- Onboarding flows
- Feature explanations
- Error pages

**Illustration Library:**
- Contractor characters (various trades)
- Home elements (houses, tools, materials)
- Map/location concepts
- Success/confirmation moments

## Iconography Style

### Icon System

**Style:**
- **Outline icons** for UI elements (navigation, actions)
- **Filled icons** for active states and solid indicators
- 2px stroke weight for outline icons
- Rounded caps and joins
- 24x24px default size, 20x20px for dense UIs

**Icon Library:**
Use Lucide icons as the base set (consistent, open-source, well-designed)

**Custom Icons Needed:**
- Trade-specific icons (plumber, electrician, roofer, etc.)
- Verification badge icon
- Map marker variants (available, busy, top-rated)
- Job status icons (open, in-progress, completed)

### Icon Usage

| Context | Style | Size | Color |
|---------|-------|------|-------|
| Navigation | Outline | 24px | Medium Gray, Blue when active |
| Buttons | Outline or Filled | 20px | Inherit button color |
| Form inputs | Outline | 20px | Medium Gray |
| Status indicators | Filled | 16px | Semantic color |
| Map markers | Custom filled | 32-48px | Primary or semantic |

### Map Marker Icons

**Standard Marker:**
- 36x48px pin shape
- Primary blue fill (`#2563EB`)
- White icon inside (trade symbol or initial)
- Subtle shadow: `0 2px 4px rgba(0,0,0,0.2)`

**Active/Hover Marker:**
- 44x56px (scaled up 120%)
- Same styling, slightly larger shadow
- Optional: pulse animation

**Cluster Marker:**
- Circular badge with count
- 40px diameter
- Blue gradient background
- White text, Bold weight

## Visual Identity Summary

> Contractie's visual identity balances professionalism with warmth. The blue-teal palette evokes trust and capability, while the coral accents add energy and approachability. Clean typography ensures readability across all devices, and authentic imagery reinforces our local, community-focused mission.

**Key Takeaways:**
1. **Blue is primary** — Use `#2563EB` for main actions and brand moments
2. **Inter everywhere** — Single font family for consistency
3. **Real photos** — Authentic imagery over stock photography
4. **Clear hierarchy** — Color and typography guide users naturally
5. **Breathing room** — Generous spacing creates a premium, uncluttered feel
